Fork Structure — Diagram Style

Three approaches to visualising the network fork history. Which direction feels right?

A — Branch Timeline

Git-log style. Main chain as a horizontal axis, each reorg shown as a branch diverging upward. Depth = branch height. Color = severity.

Most informative

B — Sankey / Peer Flow

Flows show how peers moved between consensus states across a reorg. Band width = peer count. Shows network-level impact, not chain structure.

Your idea

C — Radial Fork Map

Current height at center. Each reorg radiates outward — distance = rollback depth, angle = recency. Compact, spatial overview.
